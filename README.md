# mate-v
A small RV32I RISC-V core developed from [Steve Hoover](https://www.linkedin.com/in/steve-hoover-a44b607/) of [Redwood EDA, LLC](https://redwoodeda.com), [Linux Foundation](https://www.linuxfoundation.org/), and [RISC-V International](https://riscv.org)

## Developing

1. Install python dependencies:

```
pip install -r requirements.txt
```

2. Run makerchip-app

```
makerchip src/main.tlv
```
